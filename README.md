Este repositório centraliza meus estudos, anotações e projetos desenvolvidos durante o Ciclo Preparatório do programa **Lighthouse da Indicium**. O objetivo é documentar a jornada de aprendizado na **Modern Data Stack (MDS)** e aplicar as melhores práticas de Engenharia e Analytics Engineering.

---

## 🎯 Objetivo do Repositório
Demonstrar a evolução técnica nos pilares exigidos pela Indicium:
- **Cultura de Dados:** Mentalidade consultiva e foco na geração de valor.
- **SQL Avançado:** Modelagem e manipulação de dados complexos.
- **Modern Data Stack:** Uso de ferramentas como dbt, Airflow, Snowflake e Docker.
- **Qualidade e Documentação:** Código limpo, versionamento semântico e READMEs explicativos.

---

## 🚀 Roadmap de Estudos

### Semana 1: Fundamentos e Arquitetura
- [ ] Conceitos de OLTP vs. OLAP.
- [ ] Transição de ETL para ELT.
- [ ] Introdução ao Data Warehouse e Data Lake.
- [ ] **Lab:** Configuração de ambiente local com Docker.

### Semana 2: Engenharia com SQL & Python
- [ ] SQL Avançado: CTEs, Window Functions e Subqueries.
- [ ] Python para Dados: Extração de APIs e manipulação de JSON/Parquet.
- [ ] Versionamento de código com Git (Gitflow básico).
- [ ] **Lab:** Script de extração resiliente (Idempotência).

### Semana 3: Analytics Engineering & Modelagem
- [ ] Modelagem Dimensional (Star Schema e Snowflake Schema).
- [ ] dbt (Data Build Tool): Modelagem modular e documentação de dados.
- [ ] Visualização de Dados: Princípios de BI.
- [ ] **Lab:** Transformação de dados brutos (Raw) em tabelas de negócio (Refined).
